Frequently Asked Questions

Everything you need to know about DIF Docs

What file types does DIF Docs support? +
DIF Docs supports Word documents (.docx), Excel files (.xlsx, .csv), and text-based PDF files. Scanned documents or image-only files are not supported as they require OCR processing.
How does the credit system work? +
One credit equals one document comparison (one job). Subscription plans include a fixed number of credits per billing period. Pay-as-you-go credits never expire and accumulate over time. Subscription credits are consumed first.
Are my documents stored permanently? +
No. DIF Docs processes files temporarily and automatically deletes them after processing. We do not permanently store user documents or generated results to protect your privacy.
What happens if Word/Excel shows a warning? +
Documents generated by DIF Docs may trigger warnings in Word or Excel due to programmatic regeneration. These warnings do not indicate malware or unsafe content â€” they're normal for programmatically created files.
Can I get a refund? +
Full refunds are available within 14 days of purchase if zero credits have been consumed and no document comparisons have been performed. Once you use any credit, refunds are not available. Contact contact@studios216.com for refund requests.
